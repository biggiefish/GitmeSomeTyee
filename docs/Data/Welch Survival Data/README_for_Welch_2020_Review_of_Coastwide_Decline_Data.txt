File descriptions:

Welch_2020_Review_of_Coastwide_Decline_SAR_Data.csv-- Contains the compilation of the survival-to-adult return (SAR) data used in the publication. See Table S1 from the publication for definition of fields and citations.

Welch_2020_Review_of_Coastwide_Decline_PSC_Harvest_Data.csv-- Contains the harvest data from the Pacific Salmon Commission. Data were provided by brood year which we converted to return year (for consistency with other datasets) using the dominant year of return as brood year + 4. The harvest rate data are in column TotHR_tot which includes marine and freshwater harvest plus incidental mortality. See Fig 7 (Subyearling panel) and Table S2.

Welch_2020_Review_of_Coastwide_Decline_WA_OR_Joint_Staff_Harvest_Data.csv-- Contains the harvest data from the Washington Oregon Joint Staff Report (2019). See Fig 7 (Yearling panel) and Table S2.

Welch_2020_Review_of_Coastwide_Decline_Agency_Harvest_Data.csv-- Contains the harvest data from three hatcheries. Where necessary (Original Year Type), we converted brood year to return year using the dominant year of return as brood year + 4. See Fig 7 (Yearling panel) and Table S2.

Welch_2020_Review_of_Coastwide_Decline_Resampling.R-- R script to resample the SAR dataset 2010-2014 and generate Fig S1. To run, the SAR dataset (Welch_2020_Review_of_Coastwide_Decline_SAR_Data.csv) must be in the same directory.
